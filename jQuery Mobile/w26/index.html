PICT B.E. Course Structure

Welcome! Explore the B.E. curriculum year-wise below:

First Year (FE)

  • Engineering Mathematics-I

    Fundamentals of calculus and linear algebra.

  • Basic Electronics

    Intro to semiconductors and circuits.

  • Engineering Physics

    Mechanics, waves, and quantum concepts.

Second Year (SE)

  • Data Structures

    Stacks, queues, trees, and graphs.

  • Digital Electronics

    Logic gates, flip-flops, and circuits.

  • OOP using Java

    Classes, objects, and inheritance in Java.

Third Year (TE)

  • Operating Systems

    Processes, scheduling, and memory management.

  • Computer Networks

    Network layers, protocols, and routing.

  • Database Management Systems

    SQL, normalization, transactions.

Final Year (BE)

  • Machine Learning

    Supervised, unsupervised learning methods.

  • Cloud Computing

    Virtualization, services, deployment models.

  • Capstone Project

    Final year industrial project and report.
